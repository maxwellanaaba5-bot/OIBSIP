# OIBSIP
# Task 1: Basic Network Scanning with Nmap

## Objective
Perform a network scan using Nmap to identify open ports and services.

## Target
192.168.1.122

## Commands Used
- nmap 192.168.1.122
- nmap -sV 192.168.1.122

## Results
- Open Port: 8080/tcp
- Service: HTTP
- Version: SimpleHTTPServer 0.6 (Python 3.13.12)

## Conclusion
The scan successfully identified an active HTTP service running on port 8080.
